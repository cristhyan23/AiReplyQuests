## Respostas da Prova de Análise de Dados

Aqui estão as respostas para as 10 perguntas da prova, analisando cada uma de forma detalhada:

**1. Questão (1):** Um estudo sobre o impacto de diferentes níveis de instrução no acesso a tecnologias digitais categoriza os entrevistados em três níveis de instrução e registra o número de dispositivos tecnológicos que possuem.

   Considerando o texto, analise as afirmativas abaixo:

   I. Níveis de instrução podem ser classificados como uma variável qualitativa ordinal.

   II. A quantidade de dispositivos tecnológicos possuídos é uma variável quantitativa contínua.

   III. Variáveis qualitativas como o tipo de dispositivo tecnológico possuído não têm ordem natural.

   * **Interpretação:** A questão pede para classificar as variáveis "níveis de instrução" e "quantidade de dispositivos tecnológicos" e "tipo de dispositivo tecnológico" como qualitativas ou quantitativas, e, se qualitativas, se são nominais ou ordinais.
   * **Resposta:** Apenas I e III estão corretas.
   * **Justificativa:**
      * **I. Correta:** Níveis de instrução (ex: fundamental, médio, superior) são **qualitativos**, pois representam categorias, e **ordinais**, pois possuem uma ordem natural hierárquica (fundamental < médio < superior).
      * **II. Incorreta:** A quantidade de dispositivos tecnológicos é **quantitativa discreta**, pois representa valores inteiros e não pode assumir valores intermediários (ex: 1, 2, 3 dispositivos, mas não 2,5).
      * **III. Correta:** O tipo de dispositivo tecnológico (ex: celular, computador, tablet) é **qualitativo nominal**, pois as categorias não possuem ordem natural hierárquica.

**4. Questão (4):** Em um estudo de biologia marinha, pesquisadores estão interessados em analisar a relação entre o tamanho de diferentes espécies de tubarões e sua capacidade de migração. Para isso, coletam dados sobre o comprimento (em metros) dos tubarões e categorizam as espécies segundo sua distância de migração (curta, média e longa). Com base nessas informações, como podemos classificar a variável comprimento dos tubarões e a variável distância de migração das espécies?

   * **Interpretação:** A questão pede para classificar as variáveis "comprimento dos tubarões" e "distância de migração" como qualitativas ou quantitativas, e, se qualitativas, se são nominais ou ordinais.
   * **Resposta:** Comprimento: Quantitativa contínua; Distância de migração: Qualitativa ordinal.
   * **Justificativa:**
      * **Comprimento:** É **quantitativo contínuo**, pois pode assumir qualquer valor dentro de um intervalo (ex: 2,5 metros, 3,1 metros).
      * **Distância de migração:** É **qualitativo ordinal**, pois as categorias (curta, média, longa) possuem uma ordem natural hierárquica (curta < média < longa).

**5. Questão (5):** Um pesquisador analisa os resultados de um teste aplicado a um grupo de estudantes. Após calcular a média dos resultados, ele determina que a variância dos resultados é 16. Com base nessa informação, qual é o desvio-padrão dos resultados do teste e o que isso indica sobre a distribuição das notas dos estudantes em torno da média?

   * **Interpretação:** A questão pede para calcular o desvio padrão a partir da variância e interpretar o resultado.
   * **Resposta:** O desvio-padrão é 4, indicando que as notas se distribuem moderadamente em torno da média.
   * **Justificativa:** O desvio padrão é a raiz quadrada da variância. Neste caso, a variância é 16, então o desvio padrão é √16 = 4. 
      * Um desvio padrão de 4 indica que as notas se dispersam moderadamente em torno da média. Quanto maior o desvio padrão, maior a dispersão, e quanto menor o desvio padrão, mais concentradas as notas estão em torno da média.

**6. Questão (6):** Um analista está examinando o desempenho mensal de vendas de uma loja ao longo do último ano. Ele deseja apresentar os dados de uma maneira que facilite a visualização das tendências ao longo do tempo. Qual dos seguintes tipos de gráficos ele deveria utilizar para atender melhor a este objetivo?

   * **Interpretação:** A questão busca o tipo de gráfico mais adequado para representar dados de séries temporais (dados coletados ao longo do tempo).
   * **Resposta:** Gráfico de linhas, adequado para ilustrar variações e tendências em séries temporais.
   * **Justificativa:** Gráficos de linhas são ideais para mostrar a evolução de dados ao longo do tempo, pois ligam pontos de dados em uma linha, evidenciando tendências de crescimento, decrescimento ou estabilidade.

**7. Questão (7):** (FGV/2016) Existem dois métodos relativamente usuais para identificar, num conjunto de dados, valores não aderentes, denominados outliers. Um deles utiliza uma distribuição teórica, enquanto o outro emprega duas medidas descritivas, uma de posição e outra de dispersão. A propósito:

   * **Interpretação:** A questão discute dois métodos para identificar outliers: um baseado em uma distribuição teórica e outro baseado em medidas descritivas (posição e dispersão). 
   * **Resposta:** o método Box Plot é baseado em estatísticas de ordem.
   * **Justificativa:** O método Box Plot utiliza estatísticas de ordem para definir quartis e limites superior e inferior. Outliers são valores que excedem esses limites, indicando valores discrepantes em relação à maioria dos dados. O z-score, por outro lado, é baseado em uma distribuição teórica (normal) e utiliza a média e o desvio padrão para identificar outliers.

**9. Questão (9):** Um estatístico coleta dados sobre a quantidade de horas que os alunos de uma escola passam fazendo lição de casa por semana. Ele deseja sintetizar essas informações para representar um valor central que descreva a experiência típica dos alunos. Qual das seguintes medidas ele deveria calcular para atingir esse objetivo, considerando que os dados não estão organizados em uma distribuição de frequência?

   * **Interpretação:** A questão pede para identificar a melhor medida de tendência central para representar dados brutos (não agrupados em uma distribuição de frequência).
   * **Resposta:** Mediana, que identifica o valor do meio quando os dados são ordenados do menor para o maior número de horas.
   * **Justificativa:**
      * **Média:** É sensível a outliers (valores discrepantes), que podem distorcer o valor central.
      * **Moda:**  É útil para dados categóricos, mas não é representativa para dados quantitativos contínuos.
      * **Variância e Desvio Padrão:** São medidas de dispersão, não de tendência central.
      * **Mediana:** É a medida mais robusta em relação a outliers e representa o valor que divide o conjunto de dados em duas metades iguais.
